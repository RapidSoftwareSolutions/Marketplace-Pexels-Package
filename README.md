## Pexels.searchPhotos
Search Photos

| Field  | Type       | Description
|--------|------------|----------
| apiKey | credentials| Api KEY
| query  | String     | Query to search
| perPage| Number     | Photos data per page
| page   | Number     | Page number. For pagination
